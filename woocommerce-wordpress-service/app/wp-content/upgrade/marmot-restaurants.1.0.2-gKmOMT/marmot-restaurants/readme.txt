== Description ==

Marmot Restaurants is a fast and fully customizable theme. It works with Elementor templates. You can build almost everything you can imagine.
Theme is perfect for blog, portfolio, presentational, business website and WooCommerce storefront.
* Tens of pre-made websites available just behind a click with our one-click demo installer. 
* Build any kind of website within minutes.
* No Coding required.

== Upgrade Notice ==

Automatic updates should work smoothly, but we still recommend you back up your site.
