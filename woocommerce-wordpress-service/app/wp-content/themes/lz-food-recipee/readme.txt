For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Social Icon Section:
======================

1. Go to Customizer >> Theme Settings >> Social Icon Section, add social icon urls.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, featured image and publish the page. Repeate the same step for other slider pages.

2. Go to Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider and select the pages of slider that you have created.

For Featured Products Section:
================================

1. Add WooCommerce plugin, install & activate the plugin.

2. Go to Dashboard >> Products >> Add New, Create a new product and name it, set it's featured image, add the content, add price and publish the product. Repeat the step for other featured products.

3. Go to Dashboard >> Pages >> Add New, Create a new page and name it, add WooCommerce product shortcode "[products limit="4" columns="4"]" and publish the product.

4. Go to Customizer >> Theme Settings >> Featured Products, add section short title, section title and select the product page that you have created and publish it.

For Restaurant Menu Section:
=============================

1. Go to Dashboard >> Posts >> Category, add new category for restaurant menu section, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, set it's featured image, add the content, assign the restaurant menu category and publish the page. Repeat the step for other restaurant menu posts.

3. Go to Customizer >> Theme Settings >> Restaurant Menu, add section title and select the category that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here.
