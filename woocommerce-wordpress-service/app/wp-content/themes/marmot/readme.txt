== Description ==

Marmot is a fast and fully customizable theme. It works with Elementor templates. You can build almost everything you can imagine.
Theme is perfect for blog, portfolio, presentational, business website and WooCommerce storefront.
* Tens of pre-made websites available just behind a click with our one-click demo installer. 
* Build any kind of website within minutes.
* No Coding required.

Create amazing website fast and easy:
1. choose and import demo
2. customize templates (if you need further customizations)
3. change content
4. your website is ready to go

Other features: Drag & Drop, Responsive, Translation Ready, Custom post types integrated. Premium features like popup building system, WooCommerce optimizations and some third-party premium plugins available with our PRO Plugin.

Learn more on our website: https://marmot.hqwebs.net/marmot/
Start exploring the demos: https://marmot.hqwebs.net/demo/

* To use all theme features you have to install our free plugin "HQTheme Extra" and "Elementor Website Builder". It contains our common library and one-click demo import.
Thank you for using our products :)

== Frequently Asked Questions ==

What is Marmot?
Marmot is lightweight WordPress theme built to use Elementor page builder templates to let you customize your site like never before.

Why to choose Marmot?
Marmot is not just theme. It is whole ecosystem built to make custom WordPress sites creation smoothly with no coding.

What is HQTheme Extra?
HQTheme Extra is free plugin that adds more features to Marmot theme like One-click demo import and library for more options.

Can I use Marmot without HQTheme Extra?
Yes, but if you want to use Elementor page bulder to fully customize your site, you need HQTheme Extra.


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type 'Marmot' in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to theme setup wizard to setup theme and import premade demo site.

== Marmot Child Theme ==

There are multiple ways you can use a child theme with the Marmot Theme, Use any one method mentioned below to create a child theme or use a child theme we have created.

1. Download the child theme from Github - https://github.com/hqwebs/marmot-theme-child/releases/download/1.0.0/marmot-theme-child.zip
2. Using Plugin - https://wordpress.org/plugins/search/Child+Theme/
3. Creating child theme by yourself - https://codex.wordpress.org/Child_Themes
